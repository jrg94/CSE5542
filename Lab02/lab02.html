Ant Maze

Controls

Below you'll find various controls for this scene. They're broken up into the following sections:
  1. Mouse
  2. Keyboard
  3. Buttons
Hope you enjoy the game!

Mouse

Click and drag the mouse to rotate the current component.

Keyboard

  • w: moves component up along y-axis
  • s: moves component down along y-axis

Buttons

The following controls are used to select the various body parts of the ant for transformation.

Each subsection refers to the level of the object in the scene graph. Try grabbing a leg and pulling it off the body!

If you don't like what you've done, consider resetting the scene!
